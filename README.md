Sliderify
====

Make presentation slide. Simply.

## Instration

```
$ git clone https://github.com/yatemmma/Sliderify.git
$ cd Sliderify
```

## Usage

Edit 'sample/contents.rb'

```
$ ruby sliderify/sliderify.rb
$ open index.html
```
