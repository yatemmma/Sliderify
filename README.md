Sliderify
====

Make presentation slide. Simply.

## Instration

```
$ git clone xxxx
$ cd xxxx
$ bundle install --path vendor/bundle
```

## Usage

```
$ bundle exec rake init my_slide
```

Edit contents.rb

```
$ bundle exec rake sliderify my_slide
$ open index.html
```
